[ Installationsanweisungen für FileMaker Server ]

1. Führen Sie den folgenden Befehl aus, um FileMaker Server interaktiv zu installieren, wobei $PACKAGE_PATH den Pfad des Installationspakets bezeichnet.
   $ sudo apt install $PACKAGE_PATH/filemaker-server-VN.BN-AR.deb

2. Öffnen Sie die Datei „Claris FileMaker Server Admin Console.html“ auf Ihrem Desktop und melden Sie sich bei FileMaker Server Admin Console an.

Hinweise:

- Schritt 2 ist nur erforderlich, wenn Sie FileMaker Server auf Ubuntu Desktop installieren. Für FileMaker Server auf Ubuntu Server können Sie Admin Console von einem Remote-Computer aus über die URL https://[host]/admin-console aufrufen.
- VN, BN und AR bezeichnen die Versionsnummer, die Build-Nummer und die Architektur.
